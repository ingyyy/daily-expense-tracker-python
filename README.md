# Daily Expense Tracker

A simple command-line Python project to manage daily expenses.

## Features
- Add a new expense
- View all expenses
- Calculate total and average expense
- Clear all expenses
- Handle invalid choices
- Exit safely

## Concepts Used
- Python lists
- while loops
- if / elif / else
- enumerate()
- sum()
- list.clear()

## Example
Menu:
1. Add a new expense
2. View all expenses
3. Calculate total and average expense
4. Clear all expenses
5. Exit
